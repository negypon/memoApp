📱 アプリの特徴

💾 完全ローカル完結
サーバー不要、ネット接続不要で動作します。データはブラウザのlocalStorageに保存されます。
✏️ 自由なメモ
普段は普通のメモ帳。必要な時だけタスク化できます。
📚 複数のメモ帳
用途別にメモ帳を分けて管理できます。
📱 PWA対応
ホーム画面に追加してアプリとして使用できます。

🚀 基本的な使い方

1. メモ帳を作成

右上の「+」ボタンをタップして新規作成。すぐに編集画面に移動します。

2. メモを書く

通常のメモ帳として自由に入力できます。1行目がタイトル代わりになります。

3. ブロックに状態を付与

左側のガター（細い列）をタップすると、ブロックの状態を切り替えられます：

  • （なし） 未指定 - まだ何も決めていない
  • ▶ 通常 - タスク候補
  • ✓ 完了 - 完了したタスク

状態は 未指定 → 通常 → 完了 → 未指定 の順に循環します。

4. 完了処理

右上の「✓」ボタンをタップすると：

  • ✓マークのブロックだけが完了ログに移動
  • メモ本文からは削除される
  • ▶マークのブロックはそのまま残る

📋 応用的な使い方

管理対象外の行

-- で始まる行は管理対象外として扱われます：

-- ここから保留
買い物メモ
牛乳
卵

ガターに表示されず、完了ログにも移動しません。区切りやメモとして便利です。

カテゴリ機能

ブロック先頭行に カテゴリ名 タスク名 の形式で記述すると、完了ログに整形されて保存されます：

ea ブレイク型_XXXの改善

→ 完了ログ: ea - ブレイク型_XXXの改善

メモ帳の並び替え

メモ帳を500ms長押しすると、ドラッグして並び替えできます。

メモ帳の削除

メモ帳を左にスワイプすると、削除ボタンが表示されます。

📊 完了ログ

下部ナビの「✓ 完了ログ」タブで確認できます：

  • 全メモ帳共通で表示
  • 新しい順に並ぶ
  • タップで詳細表示
  • 「復帰」ボタンで元のメモ帳の末尾に戻せる

📲 ホーム画面に追加（推奨）

iPhone Safari

  1. Safariで index.html を開く
  2. 画面下部の共有ボタン（□↑）をタップ
  3. 「ホーム画面に追加」を選択
  4. 「追加」をタップ

Android Chrome

  1. Chromeで index.html を開く
  2. メニュー（⋮）から「ホーム画面に追加」を選択
  3. 「追加」をタップ

💾 データのバックアップ

データはブラウザのlocalStorageに保存されています。

注意: ブラウザのデータを削除するとメモも消えます。

バックアップ方法（開発者向け）

  1. ブラウザの開発者ツールを開く（F12）
  2. Console タブを開く
  3. 以下を実行： 
    console.log(localStorage.getItem('thoughtMemo'));
  4. 表示されたJSONをコピーして保存

復元方法

  1. 開発者ツールのConsoleで以下を実行： 
    localStorage.setItem('thoughtMemo', '保存したJSON');
  2. ページをリロード

🤔 よくある質問

Q: メモが消えた！

A: ブラウザのデータ削除、シークレットモード、別のブラウザを使用していないか確認してください。

Q: データはどこに保存される？

A: ブラウザのlocalStorageに保存されます。サーバーには一切送信されません。

Q: 他のデバイスと同期できる？

A: 現バージョンでは非対応です。各デバイスでローカルに保存されます。

Q: 削除したメモは復元できる？

A: メモ帳自体の削除は復元できません。完了ログからの削除は「復帰」で戻せます。

📝 設計思想

このアプリは「思考を邪魔しないメモ帳」を目指しています：

  • 普段は自由にメモを書く
  • 必要な時だけタスク化する
  • 完了したことだけがログとして残る
  • タスク管理を強制しない
