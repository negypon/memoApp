# セットアップ手順

## 📦 必要なファイル

### コアファイル
- `index.html` - メイン画面
- `styles.css` - スタイル定義
- `app.js` - アプリケーションロジック
- `storage.js` - localStorage管理
- `block.js` - ブロック状態管理

### PWA関連
- `manifest.json` - PWA設定
- `sw.js` - Service Worker
- `icon-192.png` ⚠️ 別途ダウンロードが必要
- `icon-512.png` ⚠️ 別途ダウンロードが必要

### ドキュメント
- `README.md` - 詳細ドキュメント
- `guide.html` - 使い方ガイド（ブラウザで表示可能）

## 🚀 セットアップ方法

### 1. アイコンファイルの配置

Claudeから提供された `icon-192.png` と `icon-512.png` を `F:\claude\06_MEMO\` フォルダにコピーしてください。

### 2. 動作確認（デスクトップ）

1. `F:\claude\06_MEMO\index.html` をブラウザで開く
2. メモ帳が表示されればOK
3. `guide.html` で使い方を確認

### 3. iPhone Safariでの使用

**方法A: ローカルサーバーを立てる（推奨）**

```bash
# Python がインストールされている場合
cd F:\claude\06_MEMO
python -m http.server 8000
```

その後、iPhoneのSafariで `http://あなたのPCのIPアドレス:8000` にアクセス

**方法B: ファイルをWebサーバーにアップロード**

1. 無料のWebホスティング（GitHub Pages, Netlify等）にアップロード
2. iPhoneのSafariでアクセス
3. 共有ボタン → 「ホーム画面に追加」

## 📱 ホーム画面への追加

### iPhone Safari

1. Safariで `index.html` を開く
2. 画面下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ
5. ホーム画面にアイコンが表示されます

### Android Chrome

1. Chromeで `index.html` を開く
2. メニュー（⋮）から「ホーム画面に追加」を選択
3. 「追加」をタップ

## 🔧 トラブルシューティング

### アイコンが表示されない
→ `icon-192.png` と `icon-512.png` が正しく配置されているか確認

### Service Workerが登録されない
→ `http://` または `https://` でアクセスしているか確認（`file://` では動作しません）

### データが消える
→ ブラウザのデータ削除、シークレットモードを使用していないか確認

### iPhoneで自動ズームされる
→ 設定済みです（`font-size: max(16px, 1em)`）

## 💾 バックアップ推奨

重要なデータは定期的にバックアップしてください：

1. ブラウザの開発者ツール（F12）を開く

3. 表示されたJSONをテキストファイルに保存
